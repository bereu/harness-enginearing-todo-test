# Design Operations & Workflow

How we maintain and verify the design system.

## Daily Routine

1. **Design Sync:** Verify any changes in Penpot.
2. **Token Update:** Update `docs/design/foundations/tokens.json` if needed.
3. **Implementation:** Update CSS variables in `index.css`.
4. **Validation:** Visual review of components in the browser.

## Verification Sequence

1. **Token Check:** Ensure no hardcoded hex values in new code.
2. **Responsive Check:** Test at 320px, 768px, and 1200px widths.
3. **Contrast Check:** Verify accessibility of text and badges.
4. **Naming Check:** Ensure files and component names follow rules.
