# UI Vocabulary Normalization

Standard terms used throughout the UI to ensure consistency.

## Standard Terms

| Concept         | Term to Use   | Forbidden Terms    |
| :-------------- | :------------ | :----------------- |
| Creating a Task | `Create Todo` | Add Task, New Item |
| Deleting a Task | `Delete`      | Remove, Trash      |
| Status: Doing   | `In Progress` | Processing, Active |
| Category/Tag    | `Label`       | Tag, Category      |

## Capitalization Rules

- **Buttons:** Title Case (e.g., `Add Status`).
- **Labels:** Sentence case (e.g., `To do`).
- **Headings:** Title Case for `H1`, Sentence case for `H2`.
